# MyLog 화면 정리

## 1. 인증 화면

### 1.1 로그인 화면
- 이메일 입력 필드 (8-30자)
- 비밀번호 입력 필드 (8-20자)
- 로그인 버튼
- 회원가입 링크

### 1.2 회원가입 화면
- 이메일 입력 필드 (이메일 형식, 필수)
- 이름 입력 필드 (3-30자, 필수)
- 비밀번호 입력 필드 (비밀번호 규칙, 필수)
- 회원가입 버튼
- 로그인 링크

### 1.3 소셜 로그인 화면
- Google 로그인 버튼
- Kakao 로그인 버튼
- Naver 로그인 버튼
- 이메일 로그인 링크

## 2. 게시글 화면

### 2.1 전체 게시글 목록 화면
- 게시글 카드 리스트
  - 제목
  - 작성자
  - 카테고리
  - 썸네일 이미지
  - 태그 목록
  - 작성일시
- 검색 필드 (키워드)
- 태그 필터
- 페이지네이션

### 2.2 게시글 상세 화면
- 게시글 정보
  - 제목
  - 작성자
  - 카테고리
  - 대표 이미지
  - 본문 내용
  - 태그 목록
  - 작성일시
  - 수정일시
- 댓글 섹션
  - 댓글 목록
  - 댓글 작성 폼
- 게시글 관리 (작성자만)
  - 수정 버튼
  - 삭제 버튼

### 2.3 게시글 작성 화면
- 제목 입력 필드 (5-30자, 필수)
- 카테고리 선택 (필수)
- 본문 입력 필드 (10-3000자, 필수)
- 태그 입력 (다중 선택, 필수)
- 이미지 업로드 (필수)
- 작성 버튼
- 취소 버튼

### 2.4 게시글 수정 화면
- 제목 입력 필드 (5-30자, 필수)
- 카테고리 선택 (필수)
- 본문 입력 필드 (10-3000자, 필수)
- 태그 입력 (다중 선택, 필수)
- 기존 이미지 표시
- 새 이미지 업로드 (선택)
- 수정 버튼
- 취소 버튼

### 2.5 내 게시글 목록 화면
- 게시글 카드 리스트
  - 제목
  - 카테고리
  - 썸네일 이미지
  - 태그 목록
  - 작성일시
- 검색 필드 (키워드)
- 태그 필터
- 페이지네이션

### 2.6 게시글 검색 화면
- 검색 결과 목록
  - 제목
  - 작성자
  - 카테고리
  - 썸네일 이미지
  - 태그 목록
  - 작성일시
- 검색 필드 (키워드)
- 태그 필터
- 페이지네이션

## 3. 댓글 화면

### 3.1 댓글 작성 폼 (게시글 상세 내)
- 댓글 내용 입력 필드 (5-200자, 필수)
- 작성 버튼

### 3.2 댓글 목록 (게시글 상세 내)
- 댓글 아이템
  - 작성자 이름
  - 댓글 내용
  - 작성일시
  - 수정일시
- 대댓글 목록
  - 대댓글 아이템
    - 작성자 이름
    - 대댓글 내용
    - 작성일시
- 댓글 관리 (작성자만)
  - 수정 버튼
  - 삭제 버튼
- 대댓글 작성 버튼

### 3.3 내 댓글 목록 화면
- 댓글 리스트
  - 댓글 내용
  - 작성일시
  - 수정일시
  - 관련 게시글 링크
- 페이지네이션

### 3.4 내 게시글의 댓글 목록 화면
- 댓글 리스트
  - 작성자 이름
  - 댓글 내용
  - 작성일시
  - 관련 게시글 제목
- 페이지네이션

## 4. 프로필 화면

### 4.1 내 프로필 조회 화면
- 프로필 이미지
- 이메일
- 이름
- 닉네임
- 소개글
- OAuth 제공자
- 가입일시
- 마지막 수정일시
- 프로필 수정 버튼
- 계정 삭제 버튼

### 4.2 프로필 수정 화면
- 기존 프로필 이미지 표시
- 새 프로필 이미지 업로드 (선택)
- 이름 입력 필드 (3-30자, 필수)
- 닉네임 입력 필드 (3-30자, 필수)
- 소개글 입력 필드 (최대 200자, 필수)
- 비밀번호 입력 필드 (필수)
- 수정 버튼
- 취소 버튼

- 카테고리 이름 입력 필드 (1-12자, 필수)
- 수정 버튼
- 취소 버튼

## 6. 알림 화면

### 6.1 알림 목록 화면
- 알림 리스트
  - 알림 메시지
  - 알림 타입
  - 관련 게시글 링크
  - 읽음 표시 (클릭 시 읽음 처리)
- 알림 설정 버튼
- 페이지네이션

### 6.2 알림 설정 화면
- 알림 타입 목록
  - 알림 타입 이름
  - 알림 토글 스위치 (켜기/끄기)
- 저장 버튼
